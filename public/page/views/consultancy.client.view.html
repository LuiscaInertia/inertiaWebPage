Options

4 Timeline variations (combine with .timeline) classes available:

  1. Proyectos : class="timeline" = default, 2 items per row, floating left & right, timeline runs down the middle. Is stacked on mobile.
  2. Productos : class="timeline timeline-left" = 1 item per row, floating left, timeline runs down the left with markers aligned left
  3. Servicios : class="timeline timeline-right" = 1 item per row, floating right, timeline runs down the right with markers aligned right
  4. Consultor√≠a : class="timeline timeline-stacked" = posts stack on top of each other, 1 item per row, timeline runs down the middle, markers show above posts. Is default mobile layout.
  • overlap-off = drops the item overlap
  • overlap-pull-large = class to pull item up 120px
  • overlap-pull-small = class to pull item up 30px
  • overlap-push-large = class to push item down 120px
  • overlap-push-medium = class to push item down 60px
  • overlap-push-small = class to push item down 30px
  • right = pulls item to right
  • highlight = highlights the marker with primary colour
